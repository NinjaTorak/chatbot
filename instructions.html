Instructions

HOGSbot has a few functions like telling jokes and showing memes and GIFs of which it uses APIs (Memes - Pixplorer.co.uk; GIFs - GIPHY) which have over 1000 of each. If the bot doesn't know what you're talking about, it will ask you if you want it to search the web for it, and if you agree, it will search the web for the query in a new tab. Functions: Meme Function: There is a meme function that either generates a random meme if you don't specify a meme to look for, or generates a meme about a term if you specify one. (Pixplorer.co.uk API) GIF API: See Memes ^^. Same thing, but with GIFs (GIPHY API) Jokes: For now, the bot has 9 jokes, as no clean joke API could be found. Note: This Chatbot screws up if you have a 32-bit computer.
